## 双击启动游戏后无任何反应

### 窗口打开后秒退(可以看到窗口一闪而过)

先尝试把电脑后台程序退干净, 然后再尝试看能否打开?

可能与某些程序冲突, 尝试退干净后台再启动游戏

### 卡在正在启动

确认你的设备处理器是x86处理器(台式机或笔记本), 不能是 手机 / 平板 等芯片(不支持苹果m芯片)

## 以下的在新版本中会自动开启, 一般不是下面的原因.

1. 先确认电脑的internet选项中已开启 SSL 和 TLS   (最新版本默认自动开启此项,无需手动配置)
