# 高级功能

**目前仍在测试阶段**

出于兼容性考虑默认不启用, 你可在桌面版主程序右下角的进阶设置中通过"启用高级功能" 将值设为 真 即可开启.

如有BUG[欢迎反馈](https://bug.rainysnow.com)

## 自定义清理背包

自0.96.8版本起

可在桌面版主程序右下角的进阶设置中通过"清理背包选项" 设置.

填写示例

```md
麻辣香锅-初级技能书|不绑定
铜食神宝箱菜谱碎片|绑定
```

(若该配置项留空则仍然使用旧版本的背包清理, 设置好该配置项后自动启用新清理背包, 无需其他额外设置)

该操作执行时自动完成清理, 可直接打开背包查看清理结果.

[点我查看旧版本清理列表,方便迁移](https://ini-ms.rainysnow.com/old_clean_backpag.ini)

## 自定义使用物品

填写示例

```md
十三香礼包
礼券
威望币|3
```

默认不填数字时一次最大使用20个, 填入则指定最大次数(不建议设为很大,会比较卡)

填入礼券, 则会匹配所有的,如 礼券(10D), 礼券(20D)

## 免WPE获得爆率徽章

只需要正常勾选爆率徽章, 不开WPE即可.
