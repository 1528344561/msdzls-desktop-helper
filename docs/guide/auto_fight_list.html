自定义战斗序列

TIP

在浏览器中按快捷键 CTRL + F, 然后输你想找的关键词, 可以快速定位相关内容.

在页面内按 CTRL + K, 输入相关内容, 可快速定位本文档内的相关内容, 且支持模糊搜索

自定义战斗序列适用于自动刷一连串的不同地图,方便地完成日常刷图、勇士、水晶宫、悬赏、美食大赛等需要连续挑战不同地图的场景，无需坚守在电脑前手动切换地图。

功能所在位置: 进入游戏窗口后, 点击右侧的 日常 选项卡, 下方即为自定义战斗

TIP

地图名称不要自己输, 因为地图名称需严格与游戏的地图名字相同, 否则无法正常执行

建议你手动创建那个关卡, 从房间内下方复制关卡的名字就可以了, 这样方便快捷

在两个号分开各自打的时候, 成员号需要插入一条 房主=真, 让成员号自己开房, 否则会等待其他人创建房间

除了美食里的地图名称,还支持以下额外的地图名称:

  1. 美味悬赏
  2. 火山悬赏
  3. 浮空悬赏
  4. 星际悬赏

自定义战斗支持挂机魔塔. 从房间内复制关卡的名字即可

注释

想临时不执行某些行, 直接在对应行开头加 ; 或 // 即可

修改自定义战斗默认跳转的岛屿

TIP

在 0.96.9.56版本及先前版本存在一个bug, 若没有打过一次地图, 使用跳转岛屿会导致卡在荣誉殿堂, 这个 BUG 将在 0.96.9.57 版本修复

可以通过 操作=跳转岛屿 指定跳转到哪个岛屿

操作=跳转岛屿|岛屿=美味岛|区=11

目前支持的岛屿名称: 美味岛 火山岛 浮空岛 海底 星际穿越 探险营地 沙漠 雪山雷城 奇境|

挂机相关配置

以下的必须加在地图=xxx| 后, 如 地图=古堡初探|次数=1|boss=真|秒杀=0|猫猫大法=真|延迟波次=3|浮空秒杀开关秒数=200|跨服区号=4 表示使用猫猫大法, 等待第3波后开始0秒秒杀, 200秒后重复开关浮空秒杀, 并且到跨服四区挂古堡初探地图

名称示例说明
秒杀秒杀=3000设定秒杀间隔为3000毫秒
翻牌翻牌=3设定翻牌数量为3, 不设置默认为2
卡组卡组=3设定选择第3个卡组, 不设置默认为1, 若设置为0, 则不选择任何卡组, 选择背包中的前两张卡片 (自0.96.9.30起支持)
放卡放卡=真选择 脚本放卡 而不秒杀
猫猫大法猫猫大法=真选择 开启猫猫大法 而不秒杀
延迟波次延迟波次=6不秒杀,延迟波次为6后开启秒杀 (配合放卡或猫猫大法使用, 否则无效)
延迟秒数延迟秒数=300不秒杀,延迟秒数为300后开启秒杀 (配合放卡或猫猫大法使用, 否则无效)
漫天火苗漫天火苗=5选择 漫天火苗, 设置数字代表漫天火苗持续的秒数
浮空秒杀开关秒数浮空秒杀开关秒数=300设置多少秒后开关浮空秒杀, 这样用于那些特殊BOSS种类较多的关卡, 一次可能无法秒杀所有BOSS, 需要重新开关方可秒杀.
跨服区号跨服区号=3设置挂机跨服时要跳转到的区号, 去一个人少的区将更容易组队成功 (自0.96.9.34版本起支持)
浮空秒杀开启秒数浮空秒杀开启秒数=300设置多少秒后开启浮空秒杀, 不设置默认每60秒检测一次浮空秒杀, 可用于浮空关精准定位通关时间(自0.96.9.56版本起支持)

以下的建议单开一行设置, 更清晰(单开一行后, 若需要关闭对应设置, 则再在需要的地方时候设置一句xxx=假, 否则一直保持对应功能的状态.)

名称示例说明
仅双人仅双人=真设置仅双人时开始游戏, 不填时默认不限制
房主房主=真动态调整当前账号是否为房主,假如说战斗序列里有只能单人完成的关卡比如水晶宫,这时候就可以设置 房主=真 , 让这个账号成为房主而不是加入其他人的房间
禁队友位置禁队友位置=真调整是否禁止队友位置 (自0.96.9.34版本起支持)
等待队友进入时间等待队友进入时间=0动态调整等待队友进入时间 (自0.96.9.52版本起支持)

批量配置自定义战斗

如果是新版本, 在桌面版主程序中间 点击, "打开配置工具" , 如果是旧版本桌面版, 请下载桌面版配置工具open in new window (访问密码2333) 以后, 打开, 选择软件根目录,

在自定义战斗配置区域填入相关自定义战斗方案

然后到 "批量配置" 界面

"配置名称" 填入：自定义战斗_一键签到时执行

"值" 填入：真

即可.

二级密码

一般的, 在 模拟鼠标点击 的操作过程中出现的二级密码, 你对相应账号的角色名后面加上英文括号里面写二级密码, 如提拉米苏(123456), 会在操作过程中自动输入. 并且建议不勾选桌面版右下角的 标题显示二级密码, 这样不会在软件中暴露信息

在高级功能的操作中若需要二级密码, 可以加入一个 操作=输入二级密码|密码=123456 (只对高级功能有效)

自 0.96.9.57版本起, 若 操作=输入二级密码 未填写密码, 此时自动补充操作中的密码=为你在上方提供的二级密码

遇到问题?

无法继续战斗?

如果你确定已经设置了 boss=真 但还是无法战斗

你可以把那段文本复制到记事本, 保证格式正确无误以后再粘贴回去即可.

悬赏进入的是旧地图

如果你在周四更新新地图后没有重启过桌面版,需要重启桌面版以获取最新地图信息

如果仍然进入旧地图, 你可以到官网下载最新版本.
