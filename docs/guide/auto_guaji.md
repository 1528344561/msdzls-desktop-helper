# 自动批量挂机

## 使用方法

所在位置: 桌面版主程序中右部, 选择 "自挂功能" 选项卡, 里面点击 自动挂机配置 按钮, 会打开一个新的窗口

感觉软件上写的非常明白,这部分暂时不知道该写什么

配置好 "是否房主" 这一列, 就会自动进行组队

### 配置建议

建议将奇数号设为成员, 偶数号设为房主. 这样组队成功率更高

### 每日0点自动开始上号挂机

目前暂只支持0点时自动开始日常签到, 你可以在自动日常签到界面右下角, "执行完后执行" 填入 **挂机** 即可. 然后勾选 0点时自动上号签到

这样,即便你日常里一个号也没有,也会自动进行批量上号挂机

### 挂机前自动领取温馨礼包

在新版本中, 自动批量挂机页面的**任务事项**选项卡中, 可以填入自定义操作序列

填入:

```
操作=绑定温馨礼包
操作=领取温馨礼包
```

就会执行对应的操作. 此处也支持其他操作, 详见[自定义战斗](./auto_fight_list.md#自定义操作序列)

## Q&A

Q: 顺序号的作用是什么?

A: 调整账号的顺序. 输入好顺序号以后, 单击列表头部, 自动按照从小到大顺序排序, 而非原顺序. 无其他实际意义.

Q: 发现有的号没有正确跳转到对应地图, 如设置在美味岛挂机只进了曲奇岛, 怎么办?

A: 在桌面版右下角 进阶设置, 下拉选择 "批量任务使用锁定地图" 填入 真, 点击应用配置, 然后重进游戏即可

## 自定义对局浮空秒杀(未完成)

原设定是每60秒检测一次浮空秒杀, 这对少量账号来说影响不大, 随着账号数量的增大, 尝试浮空秒杀的频率会更高, 会造成更大的CPU占用

因此在新版本中支持仅指定地图开启浮空秒杀,以降低CPU占用,尽可能登录更多账号. 格式如下:

```
茴香竹筏（日）|5|1
```

第一列为地图名称,需要严格与实际地图名称相同

第二列为在第 X 波时再开始秒杀BOSS

第三列为尝试重新开启几次浮空秒杀, **这里现在必须填1**

**你只需要添加那些你要挂的地图就行了, 不挂的那些地图不用添加. 所以工作量并不大**
