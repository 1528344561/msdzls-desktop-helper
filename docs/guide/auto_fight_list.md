# 自动战斗序列

::: info
该功能在开发中,将于0.96.1版本或0.96.2版本发布
:::

自动战斗序列适用于自动刷一连串的不同地图,方便地完成日常刷图、悬赏、美食大赛等需要连续挑战不同地图的场景，无需坚守在电脑前手动切换地图。

## 配置

每一行格式如下 (不需要加左侧数字):

```
地图=美味悬赏|秒杀=3000|次数=1
地图=雪顶火山|秒杀=3000|次数=1
```

除了美食里的地图名称,还支持以下额外的地图名称:

1. 美味悬赏
2. 火山悬赏
3. 浮空悬赏
4. 星际悬赏
5. 智能选择

地图名称需严格与map.ini中的地图相同,否则无法正常执行
