Quick Start

  1. 百度搜索下载HMCL启动器或PCL启动器
  2. 打开后,创建一个角色(用离线模式创建)
  3. 下载整合包open in new window,然后拖进启动器 (直接Download,不要点那个黄色的Install)
  4. 安装Java8open in new window,游戏Java版本选择刚才安装的Java
  5. 修复多人游戏禁止BUG, 方法一:先把网断了,然后进入游戏,再联网(最简单的方法). 方法二:修复多人游戏禁止BUGopen in new window,原文的1.16.5在这里叫 "Above and Beyond",同理也是"Above and Beyond.json"
  6. 启动游戏,多人游戏添加服务器地址

rainysnow.com

Rules

禁止生电等利用游戏BUG刷物品的行为

禁止矿透,伽马值等会对正常玩家造成不平衡的行为/模组

光影(可选)

更精美的画质和光影动效,相应地,对GPU有一定要求

下载 Oculusopen in new window 添加进mod

下载 Complementary Shadersopen in new window 放入光影文件夹
