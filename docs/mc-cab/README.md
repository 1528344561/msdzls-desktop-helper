2. 打开后,创建一个角色(用离线模式创建)
3. [下载整合包](https://www.curseforge.com/minecraft/modpacks/create-above-and-beyond/files/all?page=1&pageSize=1),然后拖进启动器 (直接Download,不要点那个黄色的Install)
4. 待mod下载完成后,启动游戏,多人游戏添加服务器IP
